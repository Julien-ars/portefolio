Colorimétrie du Cinéma

"Un ordinateur peut-il ressentir la couleur d'un film ?"
Machine Learning Web Scraping Python Octobre 2025

[1] Le Contexte

Les affiches de films jouent un rôle crucial dans le marketing cinématographique, façonnant souvent les attentes du public. Ce projet a pour but de prédire la couleur dominante d'une affiche de film en se basant uniquement sur ses métadonnées (le genre, le budget, la langue d'origine, la longueur du titre et l'année de sortie). L'enjeu est de découvrir si des données purement factuelles influencent l'identité visuelle (la couleur) choisie par les studios.

[2] La Méthodologie

1. Récolte des données : J'ai développé un script de web scraping pour extraire les données de 3000 films depuis la plateforme Hugging Face. Ces données incluent les caractéristiques classiques du film ainsi que l'extraction automatique des codes hexadécimaux des palettes de couleurs des affiches.

2. Traitement (Feature Engineering) : Plusieurs variables ont été transformées, notamment le passage au logarithme pour le budget (afin de réduire l'écrasement entre petites et énormes productions) et l'encodage des genres en variables exploitables par l'algorithme.

3. Modélisation : S'agissant d'un problème de classification multiclasse, j'ai testé plusieurs modèles algorithmiques en Python, dont la régression logistique Ridge.

[3] Les Résultats

Le modèle de régression Ridge s'est révélé être le plus performant, atteignant une précision de 34.78% sur les données de test (largement au-dessus de la probabilité aléatoire qui est d'environ 16%). Bien que modéré, ce résultat est fascinant : il démontre que l'ordinateur est capable de deviner correctement la couleur dominante d'une affiche 1 fois sur 3, sans jamais regarder l'image, prouvant que les caractéristiques invisibles d'un film dictent en partie son identité visuelle.
